About

This project was born out of doing open data work in Washington D.C. During my time in DC I knew I'd never be able to stand up any application, so I built all of my applications in HTML, CSS and JavaScript and run them on Github Pages.

While I choose Github and Github Pages to run my projects, in theory they can run anywhere, for now Github is the best vehicle possible to accomplish what I need.

While I try to keep the dependencies minimal, there are a handful of platform and code dependencies:

Platform Dependencies:

  • Github
  • Github Pages
  • oAuth.io

Code Dependencies:

  • Jquery
  • Github.js
  • SheetJS
  • Mustache

That is it. Github Pages is a perfect place to run client-side apps, and using Github + Github API via Github.js, supported by oAuth.io I'm able to use Github as the backend for the applicaiton.

I'm still heavily working on this, I will keep polishing the code and instructions on how to fork it and replicate the functionality you see here. Hopefully it will keep getting smoother!

This project is a personal project of API Evangelist, Kin Lane. If you have any comments, visit the blog, roadmap, then submit your questions via the Github Issue Management for the project.
